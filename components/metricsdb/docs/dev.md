# Using Visual Studio Code

If you are using Visual Studio Code for development, try this recommended extensions:

- Clangd
- Kylin Clangd
- C/C++ Extension Pack
- C/C++
- vscode-proto3
- YAML
- C/C++ Themes
- Yash
- Office Viewer
- CMake
- CMake Tools
